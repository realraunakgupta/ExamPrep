Pattern Master

NAS exam questions follow strict procedural patterns. Even when numerical values change, the method stays identical. Memorize these algorithms — they are universal.

100% PROBABILITY
Pattern 1: Graph Theory Matrices (A, B, Q)
+
"For the graph shown in Fig 5, consider the tree formed by branches (2, 3, 4), using this tree write incidence matrix, Tie-set matrix and cut-set matrix."
— [Q2b, May 2023] & [Q4, Mar 2025]
  1. Draw the Graph & Select a Tree.
    Replace all elements (R, L, C, sources) with line segments. Number the nodes and assign arrow directions to each branch. Select a Tree — a connected subgraph covering all nodes with no closed loops.
    Tree branches = Twigs (draw solid). Remaining = Links (draw dashed).
  2. Incidence Matrix [A] — represents KCL.
    Rows = Nodes (remove reference node for reduced matrix). Cols = Branches.
    Branch leaves node → +1
    Branch enters node → −1
    Not connected → 0
    Number of twigs = Nodes − 1 = N−1. Total branches B.
  3. Tie-set Matrix [B] — represents KVL (fundamental loops).
    Adding one link to the tree creates one fundamental loop. Rows = Loops (one per link). Cols = Branches.
    Loop direction = direction of its defining Link.
    Branch direction matches loop direction → +1
    Branch direction opposes loop → −1
    Branch not in loop → 0
  4. Cut-set Matrix [Q] — defines independent voltage variables.
    Removing one twig (and some links) divides graph in two. Rows = Cut-sets (one per twig). Cols = Branches.
    Cut-set direction = direction of its defining Twig.
    Cut link has same relative direction as twig → +1
    Cut link opposes twig direction → −1
    Branch not in cut-set → 0
95% PROBABILITY
Pattern 2: Waveform Synthesis & Laplace Transform
+
"Synthesize the waveform as shown in fig. 2" / "Obtain the Laplace transformation of the waveform given in fig.6"
— [Q1d & Q3a, May 2023] & [Q4a, Apr 2024]
  1. Identify Building Blocks (read graph LEFT to RIGHT).
    Sudden vertical jump? → Step: K · u(t − a) [K = height, a = time of jump] Slanted line starts? → Ramp: m · r(t − a) [m = slope] Slope changes? → Add new ramp to correct: slope_new = slope_old + slope_added Line returns to 0? → Subtract equivalent function to cancel
  2. Write the combined f(t).
    Traverse left to right adding a new term every time the waveform changes behavior. Example — square pulse (height 10) from t=2 to t=5: f(t) = 10·u(t−2) − 10·u(t−5)
  3. Apply Standard Laplace Transforms.
    L{ u(t−a) } = e^(−as) / s L{ r(t−a) } = e^(−as) / s² L{ sin(ωt) } = ω / (s² + ω²) L{ cos(ωt) } = s / (s² + ω²) L{ e^(−at) } = 1 / (s + a)
  4. For Periodic Waveforms (repeats every T seconds):
    Find Laplace of the first period only, call it F₁(s), then:
    F(s) = F₁(s) / (1 − e^(−Ts))
90% PROBABILITY
Pattern 3: AC Network Analysis (Mesh / Nodal / Thevenin)
+
"Find the current through 20Ω resistor using Thevenin's theorem (Fig.4)" / "Find the drop across 4Ω resistor using mesh analysis"
— [Q2a & Q1c, May 2023] & [Q2a, Apr 2024]
  1. Convert to Phasor Domain.
    Express all sources in polar form V∠θ. Convert all elements to complex impedance Z:
    Resistor → Z = R Inductor → Z = jωL Capacitor → Z = −j/(ωC) = 1/(jωC)
  2. Set Up the Mesh/Nodal Matrix.
    For Mesh: Assign clockwise currents I₁, I₂… Write KVL. Use the self-impedance shortcut:
    [Z₁₁ Z₁₂] [I₁] [V₁] [Z₂₁ Z₂₂] [I₂] = [V₂] Z_nn = sum of ALL impedances in mesh n (positive) Z_mn = negative sum of shared impedances between mesh m and n V_n = sum of voltage rises from sources, in loop direction
  3. Solve via Cramer's Rule (complex arithmetic).
    I₁ = Δ₁/Δ, I₂ = Δ₂/Δ (determinants with complex numbers) Multiplication: (a+jb)(c+jd) = (ac−bd) + j(ad+bc) Division: multiply numerator AND denominator by complex conjugate
⚠️
Calculator Warning — Most Common Source of Errors Ensure your scientific calculator is switched to Complex Mode (CMPLX) before the exam. Know how to convert between Rectangular (a+jb) and Polar (r∠θ) form instantly. Practice this beforehand — almost all calculation mistakes here come from wrong mode or wrong conjugate.

📄 Syllabus Highlights

Application of Mesh current analysis, Node voltage analysis and Network theorems in AC circuits. Graph theory: concept of tree, tie set matrix, cut set matrix and application to solve electric networks.

⚡ AC Network Theorems

Theorem What it says
Superposition Total response = sum of individual responses. Deactivate unused sources: short V-sources, open I-sources.
Thevenin's Replace network with Vth (open-circuit voltage) in series with Zth (impedance with sources off).
Norton's Replace network with IN (short-circuit current) in parallel with ZN. ZN = Zth.
Z_R = R | Z_L = jωL | Z_C = −j/(ωC)

🕸️ Graph Theory Vocabulary

Term Definition
Graph Network with all elements replaced by line segments (branches).
Node Junction where 2 or more branches meet.
Tree Connected subgraph spanning all nodes with NO closed loops.
Twig Branch belonging to the tree. Count = N − 1.
Link / Chord Branch NOT in the tree. Count = B − (N − 1).

The 3 Crucial Matrices

Incidence Matrix [A]

Encodes which branches connect to which nodes. Algebraically represents KCL at every node.

Leaves=+1 · Enters=−1 · None=0

Tie-set Matrix [B]

Describes fundamental loops — each formed by adding exactly ONE link to the tree. Represents KVL.

Same dir=+1 · Opposite=−1 · None=0

Cut-set Matrix [Q]

Describes fundamental cut-sets — each formed by removing one twig. Defines independent voltages.

Defined by Twig direction · ±1 · 0

Unit II — Waveforms & Transient Response

📄 Syllabus Highlights

Periodic waveforms and signal synthesis, properties and applications of Laplace transform of complex waveform. System modeling in terms of differential equations and transient response of R, L, C, series and parallel circuits for impulse, step, ramp, sinusoidal and exponential signals by classical method and using Laplace transform.

📈 Standard Signals & Laplace Transforms

Signal Notation L{f(t)}
Unit Impulse δ(t) 1
Unit Step u(t) 1/s
Delayed Step u(t−a) e^(−as)/s
Unit Ramp r(t)=t·u(t) 1/s²
Delayed Ramp r(t−a) e^(−as)/s²
Exponential e^(−at) 1/(s+a)
Sine sin(ωt) ω/(s²+ω²)
Cosine cos(ωt) s/(s²+ω²)
Periodic Waveform: F(s) = F₁(s) / (1 − e^(−Ts)) where F₁ = Laplace of first period only.

⏱️ Transient Response

Transients occur when a switch changes state. Energy storage elements resist sudden changes — this is the key insight.

Element What it resists At t=0⁺ acts as
Inductor (L) Change in current Open circuit
Capacitor (C) Change in voltage Short circuit (if uncharged)
INITIAL CONDITIONS (most important rule):
i_L(0⁺) = i_L(0⁻) [current through L cannot jump] v_C(0⁺) = v_C(0⁻) [voltage across C cannot jump]
Series RL — Classical Method:
V = R·i(t) + L·(di/dt) Solution: i(t) = (V/R)[1 − e^(−Rt/L)] Time constant: τ = L/R
Series RL — Laplace Method:
V/s = R·I(s) + sL·I(s) − L·i(0⁻) I(s) = V / [s(R + sL)] → Partial Fractions → i(t)
